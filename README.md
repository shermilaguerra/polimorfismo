programa de polimorfismo:  Uma empresa deseja implementar um aplicativo Java que realize seus cálculos de folha de pagamento de usando java de forma polimórfica.
Uma empresa paga seus funcionários mensalmente. 
Existem três tipos de funcionários:
 funcionários assalariados que recebem um salário mensal fixo, independentemente do número de horas trabalhadas;
 funcionários horistas que recebem um salário por hora e pagamento de horas extras por todas as horas trabalhadas que excederem 40; 
e funcionários comissionados que recebem um salário base mais uma porcentagem de suas vendas.
